### docker

1. 从容器copy文件到主机
```bash
docker cp <container id>:/entrypoint.sh .
```
2. 启动 mysql-docker
```
 docker run --name mysql-master -v /root/opt/mysql/master-cnf:/etc/mysql/conf.d -e MYSQL_ROOT_PASSWORD=123 -d dishui/mysql:5.5.4
```
